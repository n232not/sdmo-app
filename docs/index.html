Overview

Project setup

Create study projects with encounters, media slots, media types, workspace tabs, instructions, timestamp tags, and custom forms.

Review workflow

Coders open media, add timestamped labels and notes, fill out structured forms, and submit reviews when complete.

Team sync

Teams can sync project structure and review data through OneDrive, Google Drive, or a shared local/network folder.

Local media

Media files stay on each coder's computer. Each teammate links their own local copy instead of uploading video files through SDMo.

Export

Completed review data can be exported to Excel for analysis, with rows organized around encounters, media files, reviewers, timestamps, and form responses.

Open source

The code is available on GitHub for review, issue reporting, and local builds. Releases are distributed from the same repository.

Research data notice: SDMo is a tool for research workflows, not a medical device, clinical decision system, or hosted data service. Your team is responsible for appropriate consent, institutional approvals, data handling practices, and cloud-folder permissions.
